## ADDED Requirements

### Requirement: 后台统一视觉壳子
系统必须为后台管理端提供统一的驾驶舱风格页面壳子，覆盖侧边导航、顶部标题区、内容面板和全局主题配色，使管理员在不同模块间切换时保持一致的视觉体验。

#### Scenario: 打开任一后台业务页面
- **WHEN** 已登录管理员进入首页、青年信息、企业信息、招聘信息、政策管理或分析页面
- **THEN** 系统展示统一的深色驾驶舱风格页面壳子，包括左侧导航、顶部标题区、统一面板边框和一致的页面背景层次

### Requirement: 后台通用组件风格一致
系统必须对按钮、筛选栏、输入框、表格、提示信息和操作区采用统一的视觉规范，避免不同业务页面之间出现明显割裂的样式表现。

#### Scenario: 查看不同模块的列表和表单页面
- **WHEN** 管理员在不同业务模块之间切换并查看列表页、表单页或详情页
- **THEN** 系统以统一的组件风格呈现筛选区、操作按钮、表格、表单控件和提示信息，并保持一致的间距、描边、高亮和交互反馈
