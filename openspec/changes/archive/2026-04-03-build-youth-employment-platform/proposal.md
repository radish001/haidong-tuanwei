## Why

当前项目只有一个基础的 Spring Boot 示例工程，无法支撑海东市联络服务青年就业创业信息平台所需的后台登录、青年信息管理、企业招聘管理、政策发布、数据分析等核心业务流程。需要先建立一套“功能优先、样式可后置”的一体化后台平台，为后续业务录入、统计分析和运营维护提供稳定基础。

## What Changes

- 建设一套前后端一体的后台管理平台，提供登录认证、角色管理、菜单访问控制和统一后台布局。
- 新增青年信息管理模块，基于单张 `youth_info` 总表管理在校大学生、毕业未就业青年、农村社区待业青年、创业青年四类数据。
- 新增青年信息 Excel 模板下载、导入、导出能力，模板包含隐藏字典工作表和下拉选项数据源。
- 新增企业信息管理模块，采用表单方式进行新增、编辑、查询、删除，不提供 Excel 导入。
- 新增招聘信息管理模块，并与企业信息建立关联关系。
- 新增就业创业政策管理模块，支持富文本正文编辑、发布状态管理和详情查看。
- 新增首页驾驶舱和数据分析模块，基于业务数据提供汇总展示与分类统计分析。
- 新增字典与区域管理能力，为表单录入、Excel 模板和统计分析提供统一数据来源。

## Capabilities

### New Capabilities
- `admin-auth`: 后台管理员登录、会话认证、角色分配和菜单级访问控制。
- `youth-information-management`: 青年信息分类列表、查询筛选、表单维护和单表统一管理。
- `youth-information-import-export`: 青年信息 Excel 模板下载、导入校验、导入结果回执和导出。
- `enterprise-management`: 企业信息列表管理和纯表单维护能力，不包含 Excel 导入。
- `recruitment-management`: 招聘岗位信息维护，以及与企业信息的关联管理。
- `policy-management`: 政策信息 CRUD、富文本正文和发布状态管理。
- `dashboard-and-analytics`: 首页驾驶舱汇总信息和各青年分类分析页面。
- `dictionary-and-region-management`: 字典项和行政区域维护，用于表单选项、Excel 模板和统计口径统一。

### Modified Capabilities
- 无。

## Impact

- 会影响现有 Spring Boot 工程的整体结构，包括 controller、service、dao、MyBatis mapper、模板页面、静态资源和安全配置。
- 需要新增 MySQL 表结构，包括用户、角色、菜单、字典、区域、青年信息、企业信息、招聘信息、政策信息等。
- 需要引入 Spring Security、Excel 模板生成/导入、富文本编辑器等相关依赖和基础设施。
- 为后续按阶段实现后台平台、业务管理和数据分析功能建立明确的需求与设计基线。
