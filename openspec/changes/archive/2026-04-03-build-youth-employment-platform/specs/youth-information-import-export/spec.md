## ADDED Requirements

### Requirement: 青年信息导入模板下载
系统必须提供青年信息 Excel 导入模板下载能力，模板包含一个可填写主工作表和多个用于下拉选项的数据字典隐藏工作表。

#### Scenario: 下载青年信息导入模板
- **WHEN** 管理员在青年信息模块中点击下载导入模板
- **THEN** 系统下载一个包含已确认表头字段和隐藏字典工作表的 Excel 模板文件

#### Scenario: 模板包含受控下拉选项
- **WHEN** 管理员打开下载后的模板文件
- **THEN** 性别、民族、政治面貌、区域、学历等字段使用隐藏工作表提供的下拉选项数据源

### Requirement: 青年信息 Excel 导入校验
系统必须在保存前逐行校验上传的青年信息 Excel 数据，并按照导入规则拦截非法值或重复数据。

#### Scenario: 导入合法青年数据
- **WHEN** 管理员上传按模板正确填写且数据合法的 Excel 文件
- **THEN** 系统将这些记录保存到统一青年信息表中，并写入当前分类对应的 `youth_type`

#### Scenario: 返回导入失败明细
- **WHEN** 上传文件中存在非法字典值、区域无法匹配、日期格式错误、联系方式错误或重复数据
- **THEN** 系统返回逐行错误信息，至少包含失败行号和失败原因

### Requirement: 青年信息导出
系统必须允许管理员在青年信息分类页面中按当前筛选范围导出数据。

#### Scenario: 导出当前筛选结果
- **WHEN** 管理员在青年分类页面中设置筛选条件后执行导出
- **THEN** 系统下载仅包含当前分类和当前筛选结果的 Excel 文件
