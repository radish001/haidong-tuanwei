## ADDED Requirements

### Requirement: 匹配结果中的空岗位要求使用不限文案
系统 MUST 在岗位匹配学生结果页与学生匹配岗位结果页中，将招聘岗位未配置专业要求或学历要求的状态统一展示为“不限”，并与招聘岗位新增、编辑、列表筛选中的显式“不限”语义保持一致。

#### Scenario: 学生匹配岗位结果展示不限学历
- **WHEN** 管理员查看一条未配置学历要求的招聘岗位匹配结果
- **THEN** 系统 MUST 在学历要求位置展示“不限”
- **AND** 系统 MUST 不再以空白、未限制或其他不一致文案展示该状态

#### Scenario: 学生匹配岗位结果展示不限专业
- **WHEN** 管理员查看一条未配置专业要求的招聘岗位匹配结果
- **THEN** 系统 MUST 在专业要求位置展示“不限”
- **AND** 系统 MUST 不再以空白、未限制或其他不一致文案展示该状态

#### Scenario: 岗位匹配学生摘要展示不限
- **WHEN** 管理员打开一条未配置专业要求或学历要求的岗位匹配学生结果
- **THEN** 系统 MUST 在对应摘要字段展示“不限”
- **AND** 系统 MUST 与招聘岗位维护页面的显式“不限”语义保持一致
