## MODIFIED Requirements

### Requirement: 系统设置统一呈现基础数据工作台
系统 MUST 在左侧导航中继续保留“系统设置”作为一级菜单，并在进入该模块后统一呈现基础数据工作台；区域数据 MUST 作为工作台内部的固定管理内容使用，不得在当前页面层继续作为与字典管理并列强调的独立侧边子入口。对于影响多个业务模块页面展示的全局显示配置，系统 MUST 在该工作台内提供统一配置入口，且管理员 MUST 能在同一工作台中查看和维护这些配置，而不需要分别进入各业务模块单独调整。

#### Scenario: 打开系统设置模块
- **WHEN** 管理员从左侧导航进入系统设置
- **THEN** 系统 MUST 展示统一的基础数据工作台页面
- **AND** 管理员能够在该页面内继续访问公共字典、专业、学校、企业字典和区域数据等管理内容

#### Scenario: 区域数据在工作台内维护
- **WHEN** 管理员需要维护区域数据
- **THEN** 系统 MUST 在系统设置工作台内部提供区域数据入口或页签
- **AND** 当前页面层不得将区域管理继续作为独立侧边子菜单重点呈现

#### Scenario: 在系统设置中维护排序字段显示开关
- **WHEN** 管理员进入系统设置工作台并查看全局显示配置
- **THEN** 系统 MUST 提供“是否显示排序设置”开关，用于统一控制青年、企业、招聘和政策模块详情页及新增/编辑页中的“排序”字段是否展示
- **AND** 系统 MUST 持久化该开关状态，使后续进入相关业务页面时保持一致的显示行为
